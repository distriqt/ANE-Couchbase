Packagecom.distriqt.extension.couchbase
Interfacepublic interface Database extends flash.events.IEventDispatcher

This class represents a Database in the CouchbaseLite Mobile API. You should NOT create this class directly but instead use the core Couchbase extension to get or create a database.

See also

lastSequenceNumberproperty 
lastSequenceNumber:Number  [read-only]

Gets the latest sequence number used by the _DatabaseImpl. Every new Revision is assigned a new sequence number, so this property increases monotonically as changes are made to the _DatabaseImpl. This can be used to check whether the _DatabaseImpl has changed between two points in time.


Implementation
    public function get lastSequenceNumber():Number
maxRevTreeDepthproperty 
maxRevTreeDepth:int

The maximum depth of a document's revision tree (or, max length of its revision history.) Revisions older than this limit will be deleted during a compact operation. Smaller values save space, at the expense of making document conflicts somewhat more likely.

                  doc.commit();

public function getExistingDocument(documentID:String):Document

Gets the Document with the given id, or null if it does not exist.

Parameters

documentID:String — The id of the Document to get.

Returns
Document — The Document with the given id, or null if it does not exist.

Example

The following retrieves an existing document, and updates a value if the document was found. 

         var doc:Document = db.getExistingDocument( documentId );
         if (doc != null)
         {
             // Document exists
             doc.properties.someTestValue = 123456789;
             doc.commit();
         }
