Packagecom.distriqt.extension.couchbase
Interfacepublic interface Document

This class represents a Document in the CouchbaseLite Mobile API. You should NOT create this class directly but instead use the functionality of a Database to create/get a Document.

Properties should be key/value pairs. Values must be of a primitive data type, int, Number, String or Boolean

commit()method
public function commit():void

Commits any local changes to the properties object to the Couchbase database document, and updates with any revision changes.


Example

The following is a simple example showing the creation of a document and then adding some properties to the document. It assumes a database reference has already been retrieved. The properties are saved with the commit call. 

         var doc:Document = db.createDocument();
                  doc.properties.someTestValue  = 12345678;
         doc.properties.someTestString = "a test string";
                  doc.commit();
